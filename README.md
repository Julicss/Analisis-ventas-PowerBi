# 📊 Sales Data Analysis with SQL, Excel and Power BI

## 🧠 Description

This project focuses on analyzing sales data using SQL, Excel, and Power BI.

The process covers everything from database creation to building an interactive dashboard for data visualization.

---

## 🛠️ Tools Used

* SQL Server
* Excel
* Power BI

---

## 🔍 Process

1. Database and tables creation using SQL
2. Data insertion and querying
3. Data cleaning and transformation in Excel
4. Dashboard creation in Power BI

---

## 📈 Analysis

* Identification of top-selling products
* Sales analysis by month
* Total sales calculation

# 📊 Análisis de Ventas con SQL, Excel y Power BI

## 🧠 Descripción

Este proyecto consiste en el análisis de datos de ventas utilizando SQL, Excel y Power BI.

Se trabajó desde la creación de la base de datos hasta la visualización final en un dashboard interactivo.

---

## 🛠️ Herramientas utilizadas

* SQL Server
* Excel
* Power BI

---

## 🔍 Proceso realizado

1. Creación de base de datos y tablas en SQL
2. Inserción y consulta de datos
3. Limpieza y transformación de datos en Excel
4. Creación de dashboard en Power BI

---

## 📈 Análisis realizado

* Identificación de productos más vendidos
* Análisis de ventas por mes
* Cálculo del total de ventas

---

## 📸 Dashboard

![Dashboard](dashboard.png)

---

## 🚀 Conclusión

Este proyecto me permitió integrar herramientas clave del análisis de datos y comprender el flujo completo desde los datos hasta la visualización.
